git clone ----------------------------------------------создать копию рпозитоия
git pull ---------------------------------------------- скачать изменения
git checkout -b branch name --------------------------- создать ветку (branch)
git checkout branch name ------------------------------ перейти в ветку (branch) 
git add (имена файлов) -------------------------------- добафить файлы в СВОЙ репозиторий
git add. -----------------------------------------------добафить ВСЕ файлы в СВОЙ репозиторий
git commit -------------------------------------------- краткое описание внесённых изминений (открывается окно)
git commit -m "описание" ------------------------------ краткое описание внесённых изминений (приписывается сразу в консоли)
git push ---------------------------------------------- создание merge request (запрос на передачу изменений в обший репозиторий)
git status -------------------------------------------- показывает какие файлы изменены но не добавлены в СКВ
git log ----------------------------------------------- список последних версий (кто? когда? какие изменения сделал)
